# 📊 Sales Manager Power BI Dashboard

## Project Overview
This Power BI dashboard helps Sales Managers analyze business performance and make better decisions.

## Dashboard Features

- Total Sales Analysis
- Year-over-Year Sales Comparison
- Regional Performance Tracking
- Category Wise Sales Analysis
- Manager Target Achievement
- Return Analysis
- Delivery Delay Monitoring

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel / CSV

## Business Insights

This dashboard helps management:

✔ Identify top performing regions  
✔ Track manager performance  
✔ Compare actual sales vs targets  
✔ Find delayed deliveries  
✔ Reduce product returns
